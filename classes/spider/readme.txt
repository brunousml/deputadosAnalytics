Arquivos:

- phpspider/Spider.php 		: classe que realiza a busca de informações na web utilizando método post ou get.
- phpspider/SpiderPool.php	: classe que gerencia um pool de busca. Usada apenas em caso de buscas paralelas.

- GoogleSpider.php		: classe que herda da classe Spider as funcionalidades de busca remota e implementa
					os métodos específicos do spider.

- execGoogleSpider.php		: realiza o teste da classe GoogleSpider.
